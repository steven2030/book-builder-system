# Chapter Development Prompt

## Purpose
Guide the creation of individual chapters.

## Prompt Structure
```
Develop Chapter [#]:

Incorporate:
1. Plot points from master tracker
2. Character developments
3. Subplot elements
4. Timeline position
5. Tension requirements

Ensure:
- Consistent pacing
- Character voice
- Scene clarity
- Emotional impact
```

## Expected Output
1. Chapter outline
2. Scene breakdowns
3. Character moments
4. Integration points
