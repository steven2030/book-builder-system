# Character Generation Prompt

## Purpose
Create detailed character sheets and arcs.

## Prompt Structure
```
For character: [CHARACTER NAME]

Develop:
1. Backstory
2. Motivations
3. Arc progression through acts
4. Relationships
5. Key moments
6. Subplot involvement
```

## Expected Output
Complete character profile including:
1. Background
2. Personality
3. Arc development
4. Relationship map
